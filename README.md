# POJS

a simple Programming Online Judge System
精简C语言判题系统

------

## 配置 configuration

JSON格式
```json
{
    "ServerIP": "127.0.0.1",
    "ServerPort": "7777",
    "CompilerPath": ".\\POJS\\MinGW\\bin\\gcc.exe",
    "WorkPath": ".\\",
    "ResourcesPath": ".\\Resources\\"
}
```
